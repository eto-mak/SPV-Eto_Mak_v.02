# СП 70 / СП 48 — мастер-обучалка

Большой HTML-тренажер для GitHub Pages.

## Состав базы

- СП 70: **492** вопросов
- СП 48: **116** вопросов
- Всего: **608** вопросов

## Как работает

Это обучалка, не экзамен: вопрос → ответ → мгновенный результат → пояснение → следующий вопрос.

Ветки не смешиваются: одна ветка = один СП. Неверные ответы попадают в очередь повторения и показываются чаще.

## Режимы

- Все вопросы
- Только отклонения
- Контроль и документы
- Структура СП
- Ядро / точечные нормы

## GitHub Pages

1. Загрузи все файлы в корень репозитория.
2. Settings → Pages → Deploy from a branch.
3. Branch: main, folder: /root.

## Редактирование

База вопросов находится в `js/questions.js`. В одном вопросе должен быть ровно один вариант с `ok: true`.

## Ограничение

Это учебный тренажер, а не официальный текст СП. Для юридически значимых решений проверяй актуальную редакцию нормативного документа.

По СП 48 учтен нюанс Изменения №2 2025 года: часть требований по строительному контролю/исполнительной документации относится к смежному специализированному контуру СП 543.1325800.2024.
